Especificações Técnicas

📦 Lista de Produtos

41 produtos pré-configurados para contagem de estoque de cafeteria, incluindo xaropes, bebidas, ingredientes e acessórios.

🔢 Campos de Entrada

Quantidade Aberta, Quantidade Fechada e status "ACABOU" para cada produto, com validação automática.

💻 Tecnologia

HTML5, CSS3 e JavaScript Vanilla. Sem frameworks pesados, apenas código limpo e eficiente.

🌐 Compatibilidade

Funciona em todos os navegadores modernos (Chrome, Firefox, Safari, Edge) em desktop e mobile.

💾 Armazenamento

Dados salvos em localStorage do navegador com sincronização automática a cada mudança.

🚀 Performance

Carregamento instantâneo e interface responsiva, otimizada para conexões lentas.

Roadmap do Projeto

✅ Fase 1: MVP (Concluída)

Interface web responsiva, lista de produtos, campos de entrada e persistência local implementados.

🔄 Fase 2: Integração com Banco de Dados (Próxima)

Implementação de backend API, autenticação de usuários e sincronização com banco de dados centralizado.

📊 Fase 3: Relatórios e Análises

Dashboard com gráficos, histórico de contagens, alertas de produtos com baixo estoque.

👥 Fase 4: Gerenciamento de Usuários

Sistema de permissões, rastreamento de quem fez cada contagem, auditoria de alterações.

📱 Fase 5: Aplicativo Nativo

Versão mobile nativa para iOS e Android com sincronização offline-first.
